# AOP

# 概述

>    面向切面编程，利用AOP可以对业务逻辑的哥哥部分进行隔离，从而使得业务逻辑各部分之间的耦合度降低，提高程序的可重用性，同时提高了开发的效率。
>
>   不通过修改源代码的方式，在主干功能里面添加新功能

# 底层原理

>   AOP底层使用动态代理，有两种动态代理。第一，有接口情况，使用JDK动态代理；第二，无接口情况，使用CGLIB动态代理
